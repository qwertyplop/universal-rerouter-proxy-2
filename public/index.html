Setup guide

JanitorAI

  1. Open JanitorAI → Settings → API
  2. Select OpenAI as provider
  3. Paste the JanitorAI URL above into Custom API URL
  4. Put any value in the API Key field (forwarded as-is)

SillyTavern

  1. Open ST → API Connections
  2. Choose OpenAI Compatible
  3. Paste the SillyTavern URL above into Custom Endpoint (Base URL)
  4. Set your API key as needed by your upstream
Enter an upstream URL above and click Save to generate your proxy endpoint.

Settings are stored locally in your browser. Use Export to back up and Import to restore on another device.

The upstream URL is encoded in the generated link — no server-side config needed.
